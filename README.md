# BFS vs DFS - Comparaison des algorithmes sur graphes aléatoires

Projet Python pour implémenter et comparer les performances des algorithmes BFS (Breadth-First Search) et DFS (Depth-First Search) sur des graphes aléatoires.

## Description

Ce projet évalue et visualise les performances de deux algorithmes de traversée de graphe fondamentaux:
-**BFS (Breadth-First Search)**: Parcours en largeur d'abord

- **DFS (Depth-First Search)**: Parcours en profondeur d'abord

## Fichiers du projet

- `bfs_dfs.py` - Implémentation des algorithmes BFS, DFS et génération de graphes
- `example.py` - Script de benchmark et visualisation graphique des résultats
- `README.md` - Ce fichier
- `requirements.txt` - Dépendances du projet
- `LICENSE` - Licence MIT

## Fonctionnalités

- Implémentation de BFS avec calcul des distances et parents
- Implémentation itérative de DFS
- Génération aléatoire de graphes
- Mesure comparative des performances
- Visualisation graphique des résultats

## Installation

```bash
pip install -r requirements.txt
```

## Utilisation

Pour exécuter le benchmark et voir la comparaison graphique:

```bash
python example.py
```

## Résultats attendus

Le script mesure les temps d'exécution pour différentes tailles de graphes et génère un graphique comparatif.
